# Learn-Sass

Welcome to **Learn-Sass**!  
This repository is dedicated to helping you learn and master [Sass](https://sass-lang.com/), a powerful CSS preprocessor that makes writing and maintaining stylesheets easier and more efficient.

## 🚀 What is Sass?

Sass (Syntactically Awesome Style Sheets) is an extension of CSS that adds features such as variables, nested rules, mixins, functions, and more. It helps you write cleaner, more organized, and reusable CSS code.

## 📂 Repository Structure

- `scss/` — Main Sass/SCSS source files.
- `css/` — Compiled CSS output.
- `examples/` — Usage examples and sample projects.
- `README.md` — Project documentation.

## 🏁 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Abdelrahman984/Learn-Sass.git
   cd Learn-Sass
   ```

2. **Install Sass:**
   - Using npm:
     ```bash
     npm install -g sass
     ```
   - Or follow the [Sass installation guide](https://sass-lang.com/install).

3. **Compile Sass to CSS:**
   ```bash
   sass scss:css
   ```

4. **View examples:**
   - Check the `examples/` directory for sample code and demos.
